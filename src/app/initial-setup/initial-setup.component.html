Unified-RED Initial Setup

Connect to database Database Connection A database connection is required
Unified-RED requires a connection to a MongoDB or SQLite database. Postgres, MySQL, MariaDB, and Microsoft SQL Server are also supported with the installation of additional database drivers.

MongoDB (recommended)

For instructions on installing MongoDB onto your local server, see Install MongoDB â€” MongoDB Manual .

To setup a MongoDB in the cloud, see the MongoDB Atlas cloud service.

Example connection strings:

Local server: mongodb://localhost:27017/unified-red
MongoDB Atlas: mongodb+srv://kay:myRealPassword@cluster0.mongodb.net/?serverSelectionTryOnce=false&serverSelectionTimeoutMS=15000&w=majority
SQLite and others

SQLite is often included in Linux and embedded servers, however to download SQLite, see the SQLite Download Page .

Example connection strings:

Relative path sqlite:node_modules/unified-red/unified-red.db
Absolute path sqlite:/media/sdcard/data/unified-red/unified-red.db

To connect to a Postgres, MySQL, MariaDB, or Microsoft SQL Server database, use NPM to install the respective database driver.

Postgres: npm install --save pg pg-hstore
MySQL npm install --save mysql2
MariaDB: npm install --save mariadb
Microsoft SQL Server: npm install --save tedious

Example connection strings:

Postgres: postgres://user:pass@<hostname>:5432/unified-red
MySQL: mysql://user:pass@<hostname>:3306/unified-red
MariaDB: mariadb://user:pass@<hostname>:3306/unified-red
Microsoft SQL Server: mssql://user:pass@<hostname>:1433/unified-red
